This project involved mapping 12 real-world APT (Advanced Persistent Threat) campaigns to MITRE ATT&CK techniques using MISP and OpenCTI as the threat intelligence platforms. The output was a library of 47 Sigma detection rules deployed into the SIEM, providing technique-level detection coverage across the kill chain.

The work followed the threat-informed defense approach — using known adversary TTPs (Tactics, Techniques, and Procedures) to prioritize defensive investments where they have the most impact against realistic threats.

OpenCTI Setup

bash — Docker Compose 
# Clone and launch OpenCTI stack
git clone https://github.com/OpenCTI-Platform/docker.git opencti
cd opencti

# Set environment variables
cp .env.sample .env
nano .env  # Set OPENCTI_ADMIN_EMAIL, OPENCTI_ADMIN_PASSWORD

# Start all services
docker-compose up -d

# Verify services are running
docker-compose ps

Sigma Rule Development

Sigma is a generic signature format for SIEM systems. Rules written in Sigma can be converted to Splunk SPL, QRadar AQL, Elastic Query DSL, and other formats — making them portable across SIEM platforms.

yaml — Sigma Rule 
# Detect T1059.001 – PowerShell with encoded command (APT29 TTP)
title: Suspicious PowerShell Encoded Command
id: a3b9f201-1234-5678-abcd-ef0123456789
status: production
description: Detects PowerShell launched with -EncodedCommand flag
mitre_attack:
  - T1059.001
detection:
  selection:
    EventID: 4688
    CommandLine|contains:
      - "-EncodedCommand"
      - "-enc"
      - "-ec"
  condition: selection
level: high

Coverage Heatmap & Results

After rule deployment, ATT&CK Navigator was used to generate a coverage heatmap showing which techniques had active detection rules (green), partial coverage (yellow), and no coverage (red).

  • Initial Kill Chain coverage: 31% of Enterprise ATT&CK techniques
  • Post-project coverage: 58% — a 27-point improvement
  • 47 Sigma rules deployed, converted to Splunk SPL and Wazuh CDB
  • 12 high-fidelity rules generated zero false positives in first 30 days
  • Lateral movement techniques (TA0008) remain the lowest-coverage tactic — targeted next
