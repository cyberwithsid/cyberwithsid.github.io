Custom Pipeline

Built an OSINT pipeline for dark web and surface web monitoring of threat actor forums, paste sites, data breach repositories, and credential leak databases. The system monitors company-specific keywords, executive names, domain names, and email patterns — delivering automated alerts when sensitive information surfaces.

All monitoring is conducted ethically and legally, using passive observation only. No accounts were created on threat actor forums. The toolkit uses Tor for onion service access and proxied HTTPS for surface web sources.

Tor Crawler Setup

python — Tor SOCKS5 Crawler 
# Configure requests to use Tor SOCKS5 proxy
import requests
from stem import Signal
from stem.control import Controller

TOR_PROXY = {"http": "socks5h://127.0.0.1:9050", "https": "socks5h://127.0.0.1:9050"}

def rotate_tor_ip():
    with Controller.from_port(port=9051) as controller:
        controller.authenticate()
        controller.signal(Signal.NEWNYM)

def fetch_onion(url: str) -> str:
    rotate_tor_ip()
    resp = requests.get(url, proxies=TOR_PROXY, timeout=30)
    return resp.text

Keyword Matching & Alerting

python — Keyword Matcher 
# Regex-based keyword matching with context extraction
import re

KEYWORDS = [
    r"company\.com",
    r"[a-zA-Z0-9._%+-]+@company\.com",  # Email addresses
    r"CEO_Name|CFO_Name",               # Executive names
    r"\b(?:\d{4}[- ]?){3}\d{4}\b"   # PAN-like patterns
]

def scan_content(text: str, source_url: str) -> list:
    findings = []
    for kw in KEYWORDS:
        matches = re.findall(kw, text, re.IGNORECASE)
        if matches:
            findings.append({"keyword": kw, "matches": matches, "source": source_url})
    return findings
